# AIML Bootcamp 2026

This repository contains the student visualization assignment notebook for the AIML Bootcamp 2026.

## File

- `student_visualization_assignment.ipynb` - a Jupyter notebook for student performance analysis and visualization.

## Notebook Contents

The notebook includes:

- Average marks calculation with corrected loop logic.
- Student performance data using pandas.
- Basic data exploration with `shape`, `head`, `info`, and `describe`.
- Feature engineering for total score, performance percentage, attendance status, quiz status, assignment status, and eligibility.
- Matplotlib charts for attendance, scores, eligibility distribution, and attendance-vs-score comparison.

## How to Use

Open `student_visualization_assignment.ipynb` directly on GitHub, or download it and upload it to Google Colab or Jupyter Notebook.
